# dora

## 简介

dora 是一套前端监控系统，该仓库为控制台界面部分。

## 开发调试

```bash
# 克隆仓库
git clone https://github.com/dora-projects/dora-app.git

cd dora-app

# 安装依赖
yarn

# 启动服务
yarn start
```
